### Syntax

`bc = sw_nb(atomname)`

### Description



### Input Arguments

`atomName`
: String, contains the name of the atom or isotope (e.g. 13C for the isotope of carbon).

### Output Arguments

bc        Value of the bound coherent neutron scattering length  in fm.
