### Syntax

`(.cif) files or fullprof studio (.fst) files. it is able to read .cif`

### Description

 
obj = SW_IMPORT(fName, {toPlot})
 

### Input Arguments

% `fName`
: String, contain the file location and name of the .fst file.

% `toPlot`
: If true the structure will be plotted, default is false.
