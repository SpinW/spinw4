y = LORFWHM(x,p)
 
The function is normalized to integral 1 for dx=1.
 
Input:
 
x     Vector of coordinate values where the function is evaluated.
p     Parameter vector with values  p = [I Centre FWHM].
 
See also SWFUNC.GAUSS.
