### Syntax

`y = func.gaussfwhm(x,p)`

### Description

The integral of the function is normalized assuming dx = 1.
 

### Input Arguments

`x`
: Coordinate vector where the function will be evaluated.

`p`
: Parameter vector: [I Centre FWHM].
