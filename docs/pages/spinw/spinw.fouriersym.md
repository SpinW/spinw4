res = FOURIER(obj, 'option1', value1 ...)
 
Input:
 
obj           Input structure, spinw class object.
 
Options:
 
hkl           Symbolic definition of q vector. Default is the general Q
              point:
                  hkl = [sym('h') sym('k') sym('l')]
 
 
 
See also SPINW.FOURIER.
