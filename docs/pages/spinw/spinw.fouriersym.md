### Syntax

`res = fourier(obj,Name,Value)`

### Description



### Input Arguments

`obj`
: [spinw](spinw.html) object.

### Name-Value Pair Arguments

`hkl`
: Symbolic definition of q vector. Default is the general Q
  point:
      hkl = [sym('h') sym('k') sym('l')]
