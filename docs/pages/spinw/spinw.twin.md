Sub fields are:
  rotc    rotation matrices in the xyz coordinate system for
          every twin, stored in a 3 x 3 x nTwin matrix
  vol     volume ratio of the different twins, stored in a
          1 x nTwin vector
 
See also SPINW.ADDTWIN, SPINW.TWINQ, SPINW.UNIT_CELL.
