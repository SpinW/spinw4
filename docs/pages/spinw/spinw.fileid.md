FILEID(obj,fid)
 
Determines the text output of all spinw class methods. Default
is 1, where all output is printed onto the MATLAB Command
Window.
 
fidOut = FILEID(obj)
 
Outputs the stored fileID value.
 
See also SPINW, SPINW.DISPLAY.
