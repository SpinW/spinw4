### Syntax
  
`fileid(obj,fid)`
`fidOut = fileid(obj)`
  
### Description
  
`fileid(obj,fid)` determines the text output of all [spinw](spinw) class
methods. Default is 1, where all output is printed onto the MATLAB
Command Window.
   
`fidOut = fileid(obj)` outputs the stored fileID value.
