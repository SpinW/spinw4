### Syntax

`{swdescr} = display(obj)`

### Description



### Examples

crystal = spinw;
swFields = display(crystal);

### Input Arguments

`obj`
: [spinw](spinw) object.

### Output Arguments

swdescr   If output variable is given, the description of the obj object
          will be output into the swdescr variable, instead of being
          written onto the Command Window/file. Optional.
