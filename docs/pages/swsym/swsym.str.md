### Syntax

`symstr = swsym.str(symop)`

### Description



### Input Arguments

`symOp`
: Symmetry operator with rotations matrices symOp(:,1:3,:) and
  translation vectors in symOp(:,4,:).

### Output Arguments

strSym    String, contains the symmetry operations.
