### Syntax

`swplot.setrangegui(~,~,hfigure)`

### Description

This header is used for callbacks.
 
SWPLOT.SETRANGEGUI({hFigure})
 
This can be called anywhere.
 

### Input Arguments

`hFigure`
: Handle of the swplot figure. Default is the selected figure.
