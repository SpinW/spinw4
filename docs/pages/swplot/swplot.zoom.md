### Syntax

`swplot.zoom(mode, {hfigure})`

### Description



### Input Arguments

`mode`
: Either a number determining the relative zoom value, or 'auto'
  that zooms to see every object on the figure.

`hFigure`
: Handle of the swplot figure window, optional.
