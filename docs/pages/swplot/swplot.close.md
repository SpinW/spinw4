### Syntax
  
`swplot.close`
  
`swplot.close('all')`
 
`swplot.close(hFigure)`
  
### Description
  
`swplot.close` closes the active swplot figure.
   
`swplot.close('all')` closes all swplot figure.
   
`swplot.close(hFigure)` closes the swplot figure corresponding to
`hFigure` handle.
