### Syntax

`swplot.close()`

### Description

Closes the active swplot figure.
 
SWPLOT.CLOSE('all')
 
Closes all swplot figure.
 
SWPLOT.CLOSE(hFigure)
 
closes the swplot figure of the given handle (hFigure).
 

### Input Arguments

`hFigure`
: Handle of the swplot figure window.
